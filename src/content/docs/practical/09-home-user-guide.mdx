## 日常運用で知っておくべきこと

### 1. 自動更新の確認

Ubuntu Proのパッチは `unattended-upgrades` で自動適用されます。

```bash
# 自動更新が有効か確認
cat /etc/apt/apt.conf.d/20auto-upgrades

# 期待される出力:
# APT::Periodic::Update-Package-Lists "1";
# APT::Periodic::Unattended-Upgrade "1";
```

### 2. セキュリティ状態の確認

```bash
# Ubuntu Proのセキュリティ状態を確認
pro security-status

# 更新可能なパッケージの確認
apt list --upgradable
```

### 3. 定期的なメンテナンス

```bash
# 月1回程度推奨:
sudo apt update && sudo apt upgrade -y

# その後、再起動が推奨される場合:
# /var/run/reboot-required ファイルが存在するか確認
cat /var/run/reboot-required 2>/dev/null && echo "再起動が推奨されます" || echo "再起動は不要です"
```

### 4. Pro サブスクリプションの管理

```bash
# Pro の詳細状態
pro status --all

# サブスクリプション情報
pro status
# → Account と Subscription の行を確認
```

## よくある質問（FAQ）

### Q: Ubuntu Proを入れるとPCが遅くなる？

**A: いいえ。** Ubuntu Proは追加のリポジトリとLivepatchサービスを有効にするだけです。パフォーマンスへの影響はほぼゼロです。

### Q: Ubuntu Proのサブスクリプションが切れたらどうなる？

**A: すぐに危険になるわけではありません。** しかし、ESMのセキュリティ更新が受けられなくなります。個人無料プランは自動更新されるので、通常は心配不要です。

### Q: Ubuntu以外のディストリビューション（Kubuntuなど）でも使える？

**A: はい。** Kubuntu、Xubuntu、Ubuntu MATEなど、公式フレーバーでも使用できます。

### Q: 5台の制限はどう数えられる？

**A: 1つのUbuntuインスタンス（PC）= 1台** として数えられます。仮想マシンも1台としてカウントされます。

### Q: 再インストールしたらどうなる？

**A: 再度 `pro attach` コマンドでトークンを適用する**必要があります。Ubuntu Oneアカウントにログインすればトークンを確認できます。
